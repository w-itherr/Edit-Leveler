# Edit-Leveler
**Edit Leveler** is a script for Orion Drift's spectator client to mimic Orion Drift's level editor client using client sided world draw.

This script has these features:
- Building with; Cubes, Prism's and Cylinders
- Project saving and loading
- Autosaves
- Full object properties with controls; Location, Rotation, Scale, Name, Folders, Colours, more
- Even more (crazy i know!)

# Usage
## Controls
Once installed and the script is opened ingame, press F3 and place the window anywhere. Now you can start building! Start building by pressing N on your keyboard or by going over to the Assets tab on the F3 window and while hovering over the asset you press the Spacebar on your keyboard and you'll see a Cube (or another type) infront of you! Now you can use WASD and your mouse to move and turn around. To move the Object around click on one of the few lines near the Object, there will be a Red, Green and Blue line, hold one of them and move your mouse across that line and the object will move! To switch to other transform types, press either 1 (Translate), 2 (Rotate) or 3 (Scale) on your keyboard to switch between them all or go into the Controls tab and select Translate, Rotate or Scale.
## Projects
Every 30s your project will be automatically saved and be put into the Projects tab called `Autosave`! If you would like to create a new project which is **not** the auto-save, once you've built something open up the Projects tab and click Save and then input a project name and then finally click Save X objects!
To load a project just click Load and there'll be a list of projects there
